Varnish-Directadmin
===================
The scripts install Varnish Cache for Directadmin

Install instruction
===================
$ wget https://github.com/jazz1611/Varnish-Directadmin/blob/master/install.sh -O install.sh 
$ chmod +x install.sh 
$ sh install.sh
